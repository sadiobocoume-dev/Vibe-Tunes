 Installation
Clonez le dépôt
git clone https://github.com/votre-nom/vibes-tunes.git
cd vibes-tunes
Ouvrez l'application

Double-cliquez sur index.html ou

Utilisez Live Server : npx live-server

📖 Utilisation
Play/Pause - Cliquez sur le bouton central ▶️

Navigation - Utilisez les boutons ⏮️ et ⏭️

Sélection - Cliquez sur une pochette dans la liste

Recherche - Tapez dans la barre de recherche

Shuffle - Activez le mode aléatoire 🔀

Répétition - Cliquez plusieurs fois sur 🔁 pour changer de mode

Like - ❤️ pour marquer vos favoris
